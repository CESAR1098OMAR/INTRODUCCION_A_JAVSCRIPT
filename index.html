Introducción a JavaScript

let: sirve para declarar variables que pueden ser reasignadas. Estas variables tienen un alcance de bloque, lo que significa que solo son accesibles dentro del bloque de código en el que se declaran.

console.log: se utiliza para mostrar mensajes o valores en la consola del navegador. Es una herramienta útil para depurar el código y comprender su comportamiento.

typeof: para determinar el tipo de una variable, palabra clave, objeto o cadena.
